HTML form validation

The snippet of code below is written in JavaScript. It targets an HTML form element and implements validation on the fields that require text to be entered by the user before the form is submitted. If no text is entered by the user, the border of the input element turns red and an error message displays once the user clicks "submit". This code also uses regular expressions to checks the email address and telephone number input by the user to ensure they are in a valid format. You can view this code in action, in the "contact me" section of this website.
